# Design Proposals

This directory is for **in-progress design proposals** — architectural proposals that are larger than an ADR but not yet implemented.

- **ADRs** (`docs/adrs/`) record decisions already made
- **Design docs** (this directory) explore proposals that may or may not be adopted
- **Execution plans** (`docs/exec-plans/`) are implementation-level plans for accepted work

This directory is currently empty. Design docs are created when a new architectural proposal is being evaluated. For historical decisions, see `docs/adrs/`.
